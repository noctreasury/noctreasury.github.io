IP കോൺഫ്ലിക്റ്റുകൾ ഒഴിവാക്കാം
IP കോൺഫ്ലിക്റ്റ് ഒഴിവാക്കുന്നതിനായി, അനുവദിച്ചിട്ടുള്ള പുതിയ Default Gateway IP നിങ്ങളുടെ ഓഫീസിലെ മറ്റ് കമ്പ്യൂട്ടറുകൾക്ക് നൽകിയിട്ടില്ലെന്ന് ഉറപ്പുവരുത്തുക.
IP കോൺഫ്ലിക്റ്റ് എങ്ങനെ പരിശോധിക്കാം?

പുതിയ ഗേറ്റ്‌വേ IP സെറ്റ് ചെയ്യുന്നതിന് മുൻപ്, ആ അഡ്രസ് നിലവിൽ ഒഴിഞ്ഞുകിടക്കുകയാണോ എന്ന് താഴെ പറയുന്ന രീതിയിൽ പരിശോധിക്കുക.

Windows Key അമർത്തി cmd എന്ന് ടൈപ്പ് ചെയ്ത് Enter അടിക്കുക.

Reply from... : ഈ IP നിലവിൽ ഉപയോഗത്തിലുണ്ട്. ആ ഉപകരണം കണ്ടെത്തി അതിന്റെ IP മാറ്റേണ്ടതാണ്.

മൈഗ്രേഷൻ ആരംഭിക്കുന്ന സമയത്ത് :മൈഗ്രേഷൻ ആരംഭിക്കാൻ എൻ.ഒ.സി (NOC) ടീം നിങ്ങളെ സഹായിക്കുമ്പോൾ, ദയവായി ഇവിടെ മുതലുള്ള ഘട്ടങ്ങൾ പിന്തുടരുക.
അടിസ്ഥാന ഹാർഡ്‌വെയർ ക്രമീകരണങ്ങൾ
2.1 FAT PIPE ഉപകരണം
Port 1 നിലവിലുള്ള Network Switch-ലേക്ക് കണക്ട് ചെയ്യുക.
Port 2 BSNL MPLS കണക്ഷൻ (ശ്രദ്ധിക്കുക: ഇത് SDWAN ടീം നേരത്തെ കണക്ട് ചെയ്തിട്ടുണ്ടാകും).
Port 3 സെക്കൻഡറി കണക്ഷൻ (RailTel).
ചെയ്യേണ്ടത്: JUNIPER റൂട്ടറിൽ നിന്നും RailTel കേബിൾ മാറ്റി പുതിയ FAT PIPE ഉപകരണത്തിന്റെ Port 3-ൽ കണക്ട് ചെയ്യുക.
അടിസ്ഥാന സോഫ്റ്റ് വെയർ കോൺഫിഗറേഷൻ
3.1 പിസി നെറ്റ് വർക്ക് കോൺഫിഗറേഷൻ

കോൺഫിഗർ ചെയ്യുന്നതിന് മുമ്പ്, നെറ്റ് വർക്ക് ക്രമീകരണങ്ങൾ തുറക്കുക:

വിൻഡോസ് (IPv4)

  1. Win+R അമർത്തുക → ncpa.cpl ടൈപ്പ് ചെയ്യുക → Enter അമർത്തുക
  2. നിങ്ങളുടെ സജീവമായ നെറ്റ് വർക്കിൽ (ഈതർനെറ്റ്/വൈ-ഫൈ) റൈറ്റ് ക്ലിക്ക് ചെയ്ത് → പ്രോപ്പർട്ടീസ് തിരഞ്ഞെടുക്കുക
  3. ഇന്റർനെറ്റ് പ്രോട്ടോക്കോൾ വേർഷൻ 4 (TCP/IPv4) സെലക്ട് ചെയ്ത് → പ്രോപ്പർട്ടീസ് ക്ലിക്ക് ചെയ്യുക

ഇതര മാർഗ്ഗം: കൺട്രോൾ പാനൽ → നെറ്റ് വർക്ക് ആൻഡ് ഷെയറിംഗ് സെന്റർ → ചേഞ്ച് അഡാപ്റ്റർ സെറ്റിംഗ്സ്

ഉബുണ്ടു (ഡെസ്ക്ടോപ്പ്)

  1. മുകളിൽ വലതുവശത്തുള്ള നെറ്റ് വർക്ക് ഐക്കൺ (വൈ-ഫൈ/ഈതർനെറ്റ് ചിഹ്നം) ക്ലിക്ക് ചെയ്യുക
  2. ക്രമീകരണങ്ങൾ അല്ലെങ്കിൽ കണക്ഷൻ വിവരങ്ങൾ തിരഞ്ഞെടുക്കുക → വയേർഡ്/വൈ-ഫൈ ക്രമീകരണങ്ങൾ എന്നതിലേക്ക് പോകുക
  3. നിങ്ങളുടെ സജീവമായ കണക്ഷന്റെ അടുത്തുള്ള ഗിയർ ഐക്കൺ ക്ലിക്ക് ചെയ്യുക
  4. IPv4 ടാബിലേക്ക് പോകുക → ആവശ്യമായ വിവരങ്ങൾ കോൺഫിഗർ ചെയ്യുക

കമാൻഡ് ലൈൻ വഴി: nm-connection-editor അല്ലെങ്കിൽ /etc/netplan/*.yaml എഡിറ്റ് ചെയ്യുക

ക്രമീകരണം മൂല്യം
ഐപി വിലാസം മാറ്റേണ്ടതില്ല (നിലവിലുള്ളത് നിലനിർത്തുക)
സബ്നെറ്റ് മാസ്ക് മാറ്റേണ്ടതില്ല (നിലവിലുള്ളത് നിലനിർത്തുക)
ഡിഫോൾട്ട് ഗേറ്റ് വേ നിങ്ങളുടെ ട്രഷറി തിരഞ്ഞെടുക്കുക

Chrome വഴിയുള്ള പ്രോക്സി ക്രമീകരണങ്ങൾ ഒഴിവാക്കുക
പുതിയ SDWAN കണക്ഷൻ ശരിയായി പ്രവർത്തിക്കുന്നതിന് താഴെ പറയുന്ന രീതിയിൽ പ്രോക്സി സെറ്റിംഗ്‌സ് ഓഫ് ചെയ്യേണ്ടതാണ്:
1
Chrome Settings തുറക്കുക: Chrome-ന്റെ മുകളിൽ വലതുവശത്തുള്ള മൂന്ന് കുത്തുകളിൽ (⋮) ക്ലിക്ക് ചെയ്ത് Settings തിരഞ്ഞെടുക്കുക.
2
System തിരഞ്ഞെടുക്കുക: ഇടതുവശത്തെ മെനുവിൽ നിന്നും System ക്ലിക്ക് ചെയ്യുക.
3
Proxy Settings തുറക്കുക: Open your computer's proxy settings എന്നതിൽ ക്ലിക്ക് ചെയ്യുക.
4
Proxy ഒഴിവാക്കുക: Automatically detect settings എന്നതും Use a proxy server എന്നതും OFF ചെയ്യുക.
5
സേവ് ചെയ്യുക: വിൻഡോ ക്ലോസ് ചെയ്യുന്നതിന് മുൻപ് Save അല്ലെങ്കിൽ Apply ക്ലിക്ക് ചെയ്തുവെന്ന് ഉറപ്പുവരുത്തുക.
ശ്രദ്ധിക്കുക: നിങ്ങൾ ഇപ്പോൾ പുതിയ SD-WAN നെറ്റ്‌വർക്കിലേക്ക് വിജയകരമായി മാറിക്കഴിഞ്ഞു.
